# Binhex

Переписанная реализация шифровальщика от Alex0e на скретче -> https://scratch.mit.edu/projects/1062815872

Либа для шифрровки сообщений. Для правильной работы все зашифрованные сообщения 
и ключи должны быть в шестнадцатеричном виде.

## Использование

В репозитории лежат два варианта программы - написанная на C и на Python. 
Программа на C старается быть наиболее корректной в реализации, в то время 
как реализация на Python стремится быть наиболее компактной.

### C

Для того, чтобы скомпилировать прогу, достаточно ввести команду:

```sh
make
```

Это статически скомпилирует программу через gcc под musl libc. 
Если нужно использовать другой компилятор, то можно изменить нужные значения в `Makefile`.

Команда, чтобы запустить программу:

```sh
./binhex dec <КЛЮЧ> # Будет расшифровывать введёные сообщения
./binhex enc <КЛЮЧ> # Будет шифровать введёный текст
```

### Python

Команда для запуска программы:

```sh
python binhex.py dec <КЛЮЧ> # Будет расшифровывать введёные сообщения
python binhex.py enc <КЛЮЧ> # Будет шифровать введёный текст
```
